Setup Kubernetes and Skaffold

About Kubernets

Unlike in our Docker blog there are no tons of paragraphs to know about Kubernetes. Kubernetes simply exists to make it easier to organise developers application across a different number of computers. In Kubernetes Cluster we refer to these computers as Nodes. Each Node contains number of virtual machines which and each virtual machine has on an average of 4 - 6 vCPU's (Virtual CPU). Each VM has it's own vCPU's and memory.


So as we discussed in Docker Blog each container is pushed into these Nodes. I think now you have a clear understanding of how this works. If you did Great ! It's fine even if you didn't understand it at all. We are gonna talk about all these a lot in future blogs. I mean we just started right. There is a long way to go and lot to learn.

About Skaffold

Now this is important. Even though I mentioned that kubernetes eases the process of running application on different Nodes, it's still a nightmare to take care of it. Many developers choose to avoid and just deploy their application without kubernetes. That's where Skaffold comes in. Once you deploy your application to Kubernetes Cluster via Github Pull Request you don't need to use any commands to get it up and running on the cluster. Skaffold files takes care of that and it's easy to write Skaffold (.yaml) files too. Just do the hard stuff once before deployment and the next time revise your application how many ever times without issuing a single command ever again. That's the power of Skaffold.

Installation

Kubernetes.

Now for those of you who read my docker blog. You can run kubernetes on your local system with just 2 steps.
For those of you who didn't read my docker blog yet. I recommend to install and set up docker on your system. You can check my docker blog if you have any doubts.
Open up your docker desktop application, >>> Click on the settings logo you see on the top right hand side.
>>> Click on the kubernetes options appeared on the left side of your screen.
>>> Now enable Kubernetes. That's all.

For Skaffold the following.
Paste the following command in your terminals according to your system requirements mentioned.
For those who have:
Apple silicon m1 


curl -Lo skaffold https://storage.googleapis.com/skaffold/releases/latest/skaffold-darwin-amd64 
&& \ sudo install skaffold /usr/local/bin/

Apple Intel


curl -Lo skaffold https://storage.googleapis.com/skaffold/releases/latest/skaffold-darwin-amd64 
&& \ sudo install skaffold /usr/local/bin/

For Windows

Type the below command in your cmd prompt

>>> scoop install skaffold

Or

>>> choco install -y skaffold


For Linux

     && \

    && \
